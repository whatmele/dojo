## Practical usage

If you want the shortest path:

1. read [runtime-design.md](./runtime-design.md)
2. read [template-protocol.md](./template-protocol.md)
3. use `dojo template create` or `dojo artifact create` inside a workspace
4. run `dojo template lint`
